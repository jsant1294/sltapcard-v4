# SLTapCard V4

Focused product build for `card.jrtoolsusa.com`.

## Included
- Clean bilingual homepage
- One live example page: `/jrtools`
- Dashboard summary and leads board
- Lead API
- Twilio inbound webhook
- VCF download route
- Groq-ready chat route
- Local JSON fallback data store for quick testing
- Supabase-ready helper for production upgrade

## Quick start
```bash
npm install
cp .env.example .env.local
npm run dev
```
